## 4. 商店文案

### 4.1 简短描述 (Short Description)

**字数限制**: 80 字以内

```
岁时记 - 一款以中国传统节日为主题的模拟经营游戏。
在四季变换的村庄中，采集资源、制作物品、与村民交流，
体验春节、清明、端午、中秋等传统节日的魅力。
```

**字数限制**: 1500 字以内

```
【游戏简介】

岁时记是一款以中国传统节日为主题的模拟经营游戏。在这个美丽的小村庄中，

- 探索村庄周边，采集各种自然资源

- 发现稀有材料，制作珍贵物品

- 学习传统配方，制作美食和手工艺品

- 完成节日任务，准备庆典活动
- 体验特色节日小游戏

【游戏特色】

✨ 精美的国风画面
- 温馨治愈的像素风格
- 四季变换的场景设计
- 细腻的节日氛围营造

🎵 优美的背景音乐
- 古风音乐配乐
- 随季节变换的主题曲
- 沉浸式的游戏体验

📱 轻松的游戏节奏
- 适合碎片时间游玩
- 离线也能获得收益
- 无压力的游戏体验

【适合人群】

- 喜欢模拟经营类游戏的玩家
- 对中国传统文化感兴趣的玩家
- 寻找放松治愈体验的玩家
- 喜欢慢节奏游戏的玩家

快来开启你的村庄生活吧！
```
