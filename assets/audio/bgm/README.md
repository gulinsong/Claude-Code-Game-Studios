## 音乐风格

### 春季 (bgm_spring.mp3)
- 风格: 轻快、活泼、充满希望
- 乐器: 古筝、笛子、轻打击乐
- 情绪: 生机勃勃、新的开始
- BPM: 90-110

### 夏季 (bgm_summer.mp3)
- 风格: 明亮、热情、欢快
- 乐器: 琵琶、二胡、轻快节奏
- 情绪: 活力、阳光
- BPM: 100-120

### 秋季 (bgm_autumn.mp3)
- 风格: 温暖、舒缓、略带忧伤
- 乐器: 古琴、箫、柔和弦乐
- 情绪: 收获、思念、温馨
- BPM: 70-90

### 冬季 (bgm_winter.mp3)
- 风格: 宁静、纯净、略带神秘
- 乐器: 古琴、风铃、空灵音效
- 情绪: 安静、期待、温暖
- BPM: 60-80

### 节日 (bgm_festival.mp3)
- 风格: 热闹、喜庆、欢腾
- 乐器: 锣鼓、唢呐、喜庆旋律
- 情绪: 欢庆、团圆、喜悦
- BPM: 110-130

## 下载/生成来源

1. **免费素材**
   - Incompetech: https://incompetech.com/music/
   - OpenGameArt: https://opengameart.org/art-search?keys=chinese
   - Pixabay: https://pixabay.com/music/search/chinese/

2. **AI 生成**
   - Suno AI: https://suno.ai/
   - Udio: https://www.udio.com/
   - 提示词示例:
     ```
     Traditional Chinese instrumental music, guzheng and dizi,
     peaceful village atmosphere, spring morning,
     relaxing and healing, 2 minutes loop
     ```

## 音乐规格

| 属性 | 要求 |
|------|------|
| 格式 | MP3 |
| 采样率 | 44100 Hz |
| 比特率 | 128-192 kbps |
| 单文件大小 | < 2 MB |
| 声道 | 立体声 |
| 循环 | 无缝循环 |

## 文件命名规范

```
bgm_[场景/季节].mp3

Examples:
bgm_spring.mp3
bgm_festival_spring.mp3
bgm_menu.mp3
```
