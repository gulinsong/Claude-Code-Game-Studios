## Implementation Checklist

### Before Launch

- [ ] 法律审核通过
- [ ] 联系邮箱填写
- [ ] 微信公众号填写
- [ ] 隐私政策在游戏内展示
- [ ] 用户协议在游戏内展示
- [ ] 首次启动时用户同意协议

### User Flow

```
首次启动游戏
    ↓
展示用户协议
    ↓
用户点击"同意" → 进入游戏
用户点击"不同意" → 退出游戏
    ↓
游戏内设置页面
    ↓
可查看隐私政策和用户协议
```
